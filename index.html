XGen Coding Assessment

  1. Wait for Element
    1. Click the 'Get Joke' button below and after a random amount of time an element with the class 'xgen_button' will pop up in its place.
    2. Once the element is available, click on it through JavaScript. If you don't click on it in time, the button will be disabled.
    3. Bonus Points if setTimeout and/or setInterval are not used.
  2. Wait for a Global Variable to become available
    1. After the element is clicked from the first step, a joke will be generated and the answer will become available in the window.xgenVariable global variable. This will also happen after a random amount of time.
    2. Listen for when this variable (answer to the joke) is set.
    3. Bonus Points if setTimeout and/or setInterval are not used.
  3. Save answer for the next page refresh
    1. Once you have listened to the variable being set, save that value so that it can be used on the next page refresh
    2. Click on the refresh button on the top right side of your screen through JavaScript to refresh the page
    3. After the page is refreshed, the value (answer to joke) should be automatically input into the text box.
    4. Use JavaScript to click the Submit button.
